Contrataciones Abiertas

¿Para que?

Contrataciones abiertas refiere a normas y practicas para aumentar la divulgación y participación en contrataciones publicas. Incluye todo tipo de contratos, desde los básicos para adquisición de materiales hasta más complejos de obras publicas.

Estándar de Datos para Contrataciones Abiertas

Quien inicio el proceso, coalición/alianza de contrataciones, quien lo financia

Proceso de desarrollo iterativo

  • Fomentando la divulgación, datos abiertos y participación.
  • Desarrollado a partir de datos ya existentes.
  • Desarrollado a partir de las necesidades de los usuarios de los datos.
  • Prototipo modelo de datos publicado en Junio 2014 y versión 1.0 en Noviembre 2014.
  • Estándar sigue en desarrollo y se exiende con el uso del mismo.
  • Desarrollado abiertamente en https://github.com/open-contracting/standard/issues (en ingles).
